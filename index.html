Products

The Atlas Family

Four open-source products. One mission: AI that serves you — not the other way around.

Atlas Cortex

Personal AI Platform

  • Self-evolving AI assistant with 4-layer pipeline
  • Voice, chat, and CLI interfaces
  • Home automation, scheduling, media, knowledge
  • Blazing fast inference on any GPU — or CPU-only
  • OpenAI-compatible API

Atlas Sight

Accessibility Assistant

  • Vision + voice assistant for visually impaired
  • Camera scene description, OCR text reading, navigation
  • 100% offline — no subscriptions, no cloud tokens
  • Runs on phone (~1 GB RAM)
  • Voice and gesture interaction — no screen needed

Atlas CLI

Standalone AI Brain

  • Local AI coding/reasoning assistant
  • Multi-model routing (4B fast → 9B expert → 27B reasoning)
  • Code execution sandbox with self-correction
  • Session memory with crash recovery
  • No bloat — just the intelligence layer

Atlas Little Learner

Adaptive Education

  • Early education for ages 18 months to 5 years
  • 7 skill modules, adaptive difficulty
  • Toddler-friendly voice interaction
  • Frustration detection, safety limits
  • Parent dashboard

Principles

Built on convictions, not just code.

Atlas is guided by immutable principles — load-bearing walls that cannot be removed, overridden, or configured away.

🧬

Tech Specs

Built for consumer hardware. No data center required.

Component Details
Runs On Raspberry Pi Phones Laptops Desktops Workstations AMD GPUs NVIDIA GPUs Apple Silicon CPU-only
Models Qwen3.5 0.8B — Pi & phones Qwen3.5 2B — tablets & laptops Qwen3.5 4B — any 4GB+ GPU Qwen3.5 9B — expert, 8GB+ GPU Qwen3.5 27B — heavy reasoning, 20GB+
Inference llama.cpp with TurboQuant KV cache
Context Window Up to 262K tokens (model dependent)
LoRA Adapters 20+ domains — coding, medical, engineering, education, creative writing, and more
Voice Multi-provider TTS/STT — Qwen3-TTS, Fish Audio, Orpheus, Kokoro, Piper, Whisper
API OpenAI-compatible REST API on port 5100
Deployment Docker Compose stack with GPU auto-detection

Open Source Stack

Atlas Cortex is built on incredible open-source projects. Core functionality requires only llama.cpp and a Qwen model — everything else is optional and swappable.

🧠 Core (Required)

llama.cpp — Inference engine
Qwen3.5 — Language models (Apache 2.0)

🗣️ Voice (Optional)

Qwen3-TTS — Primary TTS
Fish Audio — Character voices
Piper — Ultra-fast CPU TTS
Whisper.cpp — Speech-to-text

⚡ Platform (Optional)

AMD ROCm — AMD GPU acceleration
NVIDIA CUDA — NVIDIA GPU acceleration
PEFT — LoRA adapter training
